# GameNet

**GameNet** ist ein modernes Social Network für Gamer – entwickelt von **JuGroup**.

## Idee

GameNet soll Gamer verbinden. Nutzer können Profile erstellen, Gaming-Posts teilen, Lieblingsspiele zeigen und später Communities, Clans und Events verwalten.

## Aktueller Stand

Dieses Repository enthält die erste statische GameNet-Beta-Website.

## Dateien

- `index.html` – Startseite
- `styles.css` – modernes Dark/Monochrom Design
- `script.js` – kleine Beta-Interaktion

## Geplante Features

- Gamer-Profile
- Gaming-Feed
- Lieblingsspiele
- Plattformen: PC, PlayStation, Xbox, Nintendo Switch
- Game-Communities
- Badges und Level
- GameNet Credits als internes Beta-Guthaben
- Später: Login und echte Datenbank

## Start

Öffne `index.html` im Browser.
